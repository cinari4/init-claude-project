# init-claude-project

`/init-claude-project` 슬래시 커맨드로 `CLAUDE.md`, `DEVELOPMENT.md`, `WIKI_SPEC.md` 를
현재 폴더에 생성하고, `{WIKI_ROOT}` 플레이스홀더를 입력한 위키 루트 경로로 치환한다.
이어서 위키 루트 하위에 `raw/`, `wiki/`, `assets/` 디렉토리를 만든다.

## 설치
이 저장소를 로컬 플러그인으로 추가한 뒤 Claude Code에서 활성화한다.

## 사용
대상 폴더에서 `/init-claude-project` 를 실행하고, 안내에 따라 위키 루트 절대경로를 입력한다.
